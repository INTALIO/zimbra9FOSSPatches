Marquer des mails comme non lus

Les mails que vous ouvrez sont marqués comme lus et leur attribut gras est remplacé par l'attribut normal.

  1. Cochez les cases pour sélectionner les mails à marquer comme non lus.

  2. Dans le menu déroulant Actions, choisissez Marquer comme non lu. Les mails sélectionnés sont mis en gras et la case est désélectionnée.

Remarque : Pour marquer tous les mails d'une conversion comme non lus, choisissez la conversation. Pour marquer un mail individuel d'une conversation comme non lu, ouvrez la conversation et sélectionnez le mail.

Vous pouvez utiliser le même procédure pour marquer des mails non lus comme étant lus.
