連続項目のすべての会議を削除

  1. 予約を右クリックし、[連続項目]を選択します。

  2. [連続項目を削除]を選択します。

  3. カレンダー（過去の日付と未来の日付の両方）から会議のすべての定期的な予定を削除するか、この当日以降のすべての参照を削除するかを選択します。[はい]をクリックします。

  4. [メッセージの編集] をクリックして参加者に送信するメールを編集します。[送信]をクリックしてキャンセルのメッセージを送信します。

カレンダーからその予約がすぐに削除されます。メールが参加者に送られ、その予約が各自のカレンダーから削除されます。削除を元に戻すことはできません。
