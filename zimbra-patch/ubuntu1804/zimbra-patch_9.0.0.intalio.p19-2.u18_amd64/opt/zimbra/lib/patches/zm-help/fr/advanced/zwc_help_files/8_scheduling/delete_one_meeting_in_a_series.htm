Suppression d'une réunion dans une série

  1. Cliquez avec le bouton droit de la souris sur le rendez-vous, puis sélectionnez Instance.

  2. Sélectionnez Supprimer instance.

  3. Lors de la suppression d'une instance, cliquez sur Modifier un message pour modifier le mail envoyé aux participants. Cliquez sur Envoyer pour envoyer le message d'annulation.

Le rendez-vous est immédiatement supprimé de votre calendrier. Un mail est envoyé aux participants et le rendez-vous est supprimé de leurs calendriers.
