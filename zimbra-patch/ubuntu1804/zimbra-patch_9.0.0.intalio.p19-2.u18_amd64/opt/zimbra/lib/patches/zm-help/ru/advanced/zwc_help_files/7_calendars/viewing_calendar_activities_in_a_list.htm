Просмотр мероприятий ежедневника в списке

В виде «Список» можно посмотреть сведения, недоступные в других видах ежедневника. Можно изменить способ отображения информации.

В виде «Список» можно выполнять следующие действия.

•      Отображать встречи в определенном диапазоне дат. Настроить даты«С» и «До» в текстовых полях отображения встреч.

•      Щелкнуть заголовок столбца Ежедневник, чтобы упорядочить встречи по отдельным ежедневникам.

•      Щелкнуть заголовок столбца состояния, чтобы посмотреть встречи по состоянию.

•      Щелкнуть заголовок столбца «Тема», чтобы посмотреть темы в алфавитном порядке.

•      Установите флажки возле нескольких встреч для одновременной обработки нескольких сообщений. Щелкните правой кнопкой мыши и выберите необходимое действие.
