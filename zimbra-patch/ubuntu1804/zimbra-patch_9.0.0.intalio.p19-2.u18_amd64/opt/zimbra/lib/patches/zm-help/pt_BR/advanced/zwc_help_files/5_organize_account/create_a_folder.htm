Criar uma pasta

Você pode criar uma pasta definida pelo usuário independente ou uma pasta dentro de outra pasta, incluindo as pastas de sistema.

Não é possível criar duas pastas com o mesmo nome dentro da mesma pasta principal. Por exemplo, não é possível ter duas pastas de nome Compras dentro da pasta Caixa de entrada, mas é possível ter uma pasta com nome Compras  dentro da sua Caixa de entrada e outra com o mesmo nome na pasta Enviadas.

  1. Vá para E-mail>página Pastas e clique no ícone de engrenagem Pastas.

  2. Selecione Nova pasta.

  3. Insira o novo Nome da pasta e selecione uma Cor para a pasta.

    4. Os nomes de pastas podem ter qualquer caractere, exceto dois-pontos (:), barra simples
    (/) e aspas (").

    Você tem a opção de escolher assinar o conteúdo RSS/ATOM.

  4. Na árvore Pastas, selecione o posicionamento da nova pasta.

    5. •    Clique em Pastas para exibir a nova pasta no topo da lista.

    •    Clique em uma pasta existente para posicionar a nova pasta dentro dessa pasta.

  5. Clique em OK.
