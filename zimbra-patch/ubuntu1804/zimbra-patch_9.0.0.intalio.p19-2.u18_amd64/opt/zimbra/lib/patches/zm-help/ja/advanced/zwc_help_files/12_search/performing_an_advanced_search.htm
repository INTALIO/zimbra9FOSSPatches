高度な検索を実行する

検索フィールドにキーワードや条件を入力する代わりに、高度な検索のページでは、検索フィルターを選択してより詳細な検索条件を追加することができます。

  1. [検索]ボックスで、虫眼鏡をクリックします。高度な検索のページが表示されます。

  2. ナビゲーションペインで、［高度なフィルター］リンクを選択して検索を作成します。

ページ上部の検索フィールドに検索構文が表示されます。任意のフィルターを選択できます。フィルターに新しい検索要件を追加すると、検索が実行され、結果が更新されます。

And検索およびOR検索条件を追加すれば、検索条件を絞り込むことができます。

  1. [検索]をクリックします。
