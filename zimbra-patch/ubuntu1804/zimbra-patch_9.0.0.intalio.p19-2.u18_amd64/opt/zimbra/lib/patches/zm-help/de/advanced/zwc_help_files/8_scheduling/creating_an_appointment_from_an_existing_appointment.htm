Termin aus einem vorhandenen Termin planen

Sie können einen Termin auch auf der Basis bereits vorhandener Termine anlegen. Die bereits festgelegten Teilnehmer, Orte, Notizen und Wiederholungsintervalle können auf diese Weise schnell für neue Tage und Uhrzeiten konfiguriert werden. Sie können die Angaben auch ändern, bevor Sie die Einladungen senden.

  1. Klicken Sie mit der rechten Maustaste auf den Termin, der kopiert werden soll, und wählen Sie Eine Kopie erstellen aus. Die Seite zum Erstellen eines Termins wird mit allen Informationen der kopierten Besprechung geöffnet.

  2. Wenn Sie einen Termin anlegen möchten, der sich vom kopierten Termin nur durch das Datum unterscheidet, brauchen Sie nur das Anfangs- und Enddatum zu ändern.

  3. Nehmen Sie alle weiteren gewünschten Änderungen an Teilnehmern, Orten, am Text usw. vor.

  4. Klicken Sie auf Senden, um eine Einladung an die Teilnehmer zu senden. Der Termin wird in Ihren Kalender eingetragen.
