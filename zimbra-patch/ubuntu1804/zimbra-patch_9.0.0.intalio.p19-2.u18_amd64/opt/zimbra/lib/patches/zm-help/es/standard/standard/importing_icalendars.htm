Importar agendas iCalendar

Puedes importar agendas iCalendar (iCal) guardadas con el formato .ics. Estas agendas son el estándar para el intercambio de datos de este tipo por Internet y el formato .ics es el formato estándar para los datos de estas agendas.

Puedes importar una agenda iCal a una agenda existente o crear una nueva agenda para luego importar ahí el archivo .ics.

Para importar una agenda iCal:

  1. Selecciona la pestaña Agenda.

  2. En el panel de resumen de la agenda, haz clic en Editar

  3. Selecciona una agenda de la lista o crea una nueva para recibir el archivo .ics importado y haz clic en Editar.

  4. En el campo Importar de .ICS, busca el archivo .ics que deseas importar.

  5. Haz clic en Importar.

  6. Haz clic en Cerrar.
