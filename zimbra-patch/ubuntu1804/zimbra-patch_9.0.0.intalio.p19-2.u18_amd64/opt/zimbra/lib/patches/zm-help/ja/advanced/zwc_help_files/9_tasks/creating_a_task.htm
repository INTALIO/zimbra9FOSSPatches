タスクを作成する

新しいタスクは、タスクリストにタスク名を入力して作成するか、新しいタスクページを開き、備考や添付ファイルなどを含め、タスクに関する詳細情報を追加して作成できます。

タスクをすばやく追加

  1. [タスク]ページに移動し、概要ペインでタスクを追加するタスクリストを選択します。

  2. 内容ペインの[新しいタスクを追加するには、ここをクリック]テキストボックスに、新しいタスクの名前を入れます。 

  3. Enterキーを押します。こうすることでタスクの件名がタスクリストに追加されますが、詳細は設定されていません。詳細を追加するには、タスクを編集します。

備考：  間違ったリストにタスクを作成した場合は、そのタスクを開き、[タスクリスト]ドロップダウンから正しいリストを選択するか、内容ペインから正しいリストにそのタスクをドラッグアンドドロップします。
