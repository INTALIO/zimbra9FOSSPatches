Für mich freigegebene Objekte finden

In einigen Fällen kann es vorkommen, dass Objekte für Sie freigegeben wurden, Sie jedoch keine Benachrichtigung darüber erhalten haben. Oder Ihre Organisation hat Objekte für die Öffentlichkeit freigegeben, und Sie möchten wissen, welche dies genau sind. Der Link Freigaben suchen hilft Ihnen, die Objekte zu finden, die für Sie freigegeben wurden.

Sie finden den Link Freigaben suchen in der Übersicht am Ende des Abschnitts Ordner.

  1. Klicken Sie auf Freigaben suchen.

  2. Wählen Sie im Dialogfeld "Freigegebenen Ordner hinzufügen" das freigegebene Objekt aus, das Sie abgelehnt haben, oder geben Sie im Textfeld Freigaben suchen eine Mailadresse ein, um zu sehen, ob ein Objekt von dieser Adresse für Sie freigegeben wurde.

  3. Wählen Sie im Dropdown-Menü Einschließen die Anwendung aus, nach der Sie suchen möchten. Mit der Option Alle Anwendungen können Sie in allen Anwendungen nach freigegebenen Objekten suchen.

  4. Klicken Sie auf Suchen.

  5. Wählen Sie die Objekte aus, deren Freigaben Sie annehmen möchten.

  6. Klicken Sie auf Hinzufügen.
