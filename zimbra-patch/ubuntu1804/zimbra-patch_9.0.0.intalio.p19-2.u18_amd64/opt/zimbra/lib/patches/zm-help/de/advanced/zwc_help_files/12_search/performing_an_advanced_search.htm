Erweiterte Suche durchführen

Statt im Feld "Suchen" das Schlüsselwort und Kriterien einzugeben, können Sie auf der Seite für die erweiterte Suche Suchfilter auswählen, um detailliertere Suchkriterien hinzuzufügen.

  1. Klicken Sie im Feld "Suchen" auf die Lupe. Die Seite für die erweiterte Suche wird angezeigt.

  2. Wählen Sie im Navigationsfenster Links für "Erweiterte Filter" aus, um die Suche zu erstellen.

Im Feld "Suchen" oben auf der Seite wird die Suchsyntax angezeigt. Sie können beliebige Filter auswählen. Wenn Sie den Filtern neue Suchanforderungen hinzufügen, wird die Suche ausgeführt und die Ergebnisse werden aktualisiert.

Sie können die Suchkriterien zusätzlich verfeinern, indem Sie AND- und OR-Suchbedingungen hinzufügen.

  1. Klicken Sie auf "Suchen".
