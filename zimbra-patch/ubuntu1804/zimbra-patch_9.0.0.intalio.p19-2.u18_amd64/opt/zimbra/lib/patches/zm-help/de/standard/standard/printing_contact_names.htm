Kontaktnamen drucken

Sie können ein vollständiges Adressbuch oder nur die Informationen für ausgewählte Kontakte drucken.

  • Zum Drucken ausgewählter Kontakte aus dem Adressbuch aktivieren Sie das Kontrollkästchen neben den Kontakten. Klicken Sie dann auf Drucken.

  • Um alle Kontakte im Adressbuch zu drucken, aktivieren Sie das Kontrollkästchen in der Zeile Name. Dadurch werden alle Kontakte im Adressbuch ausgewählt. Klicken Sie dann auf Drucken.
