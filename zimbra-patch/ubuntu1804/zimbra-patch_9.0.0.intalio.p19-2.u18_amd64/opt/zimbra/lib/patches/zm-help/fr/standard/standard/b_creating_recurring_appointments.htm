Créer des réunions récurrentes

Les réunions récurrentes sont des rendez-vous qui se répètent en vertu d'une certaine forme de planification. Vous pouvez programmer des réunions récurrentes pour qu'elles se répètent de manière quotidienne, hebdomadaire, mensuelle ou annuelle. De plus, vous pouvez personnaliser des planifications de rendez-vous récurrents.

  1. Pour créer un nouveau rendez-vous, cliquez sur Répétition.

  2. Dans la sectionBasique>Répétition, cliquez sur la case d'option pour sélectionner la fréquence à laquelle vous souhaitez répéter la réunion.

  3. Si la réunion récurrente dispose d’une date de fin, choisissez dans la section Avancé>Fin de finaliser les réunions après un nombre spécifique de réunions ou à une date spécifique.

  4. La section Avancé vous permet de programmer plus de modèles spécifiques.

  • Quotidienne. Personnalisez les répétitions quotidiennes pour les configurer chaque jour, ou avec une autre fréquence (par exemple tous les deux ou quatre jours).

  • Hebdomadaire. Personnalisez les répétitions hebdomadaires en choisissant s'il s'agit d'un rendez-vous récurrent hebdomadaire ou d'un rendez-vous suivant une autre configuration hebdomadaire, tous les quinze jours par exemple. Vous pouvez organiser la réunion un jour de la semaine spécifique.

  • Mensuelle. Personnalisez les répétitions mensuelles en choisissant s'il s'agit d'un rendez-vous récurrent mensuel ou d'un rendez-vous suivant une autre configuration mensuelle, tous les deux mois par exemple. Un calendrier s’affiche et vous pouvez sélectionner le ou les jours pour lesquels vous souhaitez que la réunion soit programmée.

  • Annuelle. Personnalisez les répétitions annuelles pour choisir le mois et la date.

  1. Cliquez sur Terminé pour revenir à la page de rendez-vous.
