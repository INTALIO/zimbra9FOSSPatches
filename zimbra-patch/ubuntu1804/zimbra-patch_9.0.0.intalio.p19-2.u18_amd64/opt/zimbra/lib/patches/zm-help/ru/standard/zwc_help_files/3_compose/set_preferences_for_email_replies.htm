Настройка параметров для отправки ответов на сообщения электронной почты

Чтобы ответить на сообщение в том же формате, в котором оно получено, выберите параметр Ответ/пересылка в формате исходного сообщения. Так можно обеспечить, что получатель получит сообщение в наиболее удобном для чтения формате.

  1. Перейдите на страницу Настройки>Почтав раздел Создание сообщений.

  2. Чтобы ответить на сообщение в том же формате, в котором оно получено, выберите параметр Ответ/пересылка в формате исходного сообщения.

  3. Укажите, нужно ли включать исходное сообщение в ответное. Если исходное сообщение включается в ответ:

    4. •   укажите, следует ли обозначать предыдущие ответы каким-либо символом или включать исходные сведения в заголовок (поля «Кому», «От» и «Тема»). Также можно включить исходное сообщение в качестве вложения;

    •    если необходимо поставить префикс перед предыдущими ответами, выберите либо угловую скобку (>), либо вертикальную черту (|).

  4. Нажмите кнопку Сохранить.
