Dateien zu den Ordnern "Dateien" hinzufügen

Durch das Hochladen einer Datei wird diese von Ihrem Computer in Ihren Account kopiert. Sie können dann immer auf die Datei zugreifen, wenn Sie sich bei Ihrem Account anmelden. Dateien können wie folgt hochgeladen werden:

•      Durch Klicken auf Anhänge, die Sie in Mails erhalten.

•      Mit der Funktion zum Hochladen von Dateien im Ordner "Dateien".

•      Mit den WebDav-Clientfunktionen
