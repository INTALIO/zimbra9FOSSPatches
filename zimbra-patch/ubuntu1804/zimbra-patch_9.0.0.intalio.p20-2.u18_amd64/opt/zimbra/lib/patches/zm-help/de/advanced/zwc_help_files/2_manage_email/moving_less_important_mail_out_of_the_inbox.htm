Weniger wichtige Mails automatisch aus dem Posteingang verschieben

Sie können den Filter für Aktivitätenstreams so einrichten, dass weniger wichtige Mails, z. B. Newsletter und Benachrichtigungen von sozialen Netzwerken, im Ordner "Aktivitätenstream" und nicht im Posteingang gespeichert werden. Der Ordner "Aktivitätenstream" wird der Liste Ihrer persönlichen Ordner im Übersichtsfenster hinzugefügt.

  1. Klicken Sie in der Kopfzeile Ordner des Navigationsfensters auf das Zahnradsymbol und wählen Sie Aktivitätenstream aus.

    2. Sie können dies auch auf der Seite Einstellungen > Filter über den Link Einstellungen für Aktivitätenstreams einrichten.

  2. Wählen Sie im Dialogfeld Aktivitätenstream die Option Verschieben von Mails in Aktivitätenstream aktivieren aus.

  3. Wählen Sie aus, welche Mailtypen verschoben werden sollen. Die folgenden Mailtypen können verschoben werden:

    4. •    Mails, deren Feld An oder An oder CC (Kopie an) nicht Ihre Adresse enthält

    •    Mails von Personen, die nicht in Ihrem Adressbuch enthalten sind

    •    Mails, die an einen Verteiler gesendet wurden, bei dem Sie Mitglied sind

    •    Kommerzielle Massen-Marketingmails und Bulk-Mails

  4. Klicken Sie auf OK.

  5. Klicken Sie im angezeigten Informationsfeld auf Ja.

Weitere Informationen finden Sie unter "Filterung".
