Mit "Anwesenheit" den Anwesenheitsstatus anzeigen

Hinweis: Diese Funktion ist nur für ZCS Network Edition vorgesehen und nur im erweiterten Webclient verfügbar.

(Nur Cisco-Clients)

Mit dem Cisco Jabber-Client können Sie umfassende Informationen zu Ihrer Anwesenheit und der Anwesenheit von Kontakten anzeigen

Als Anwesenheitsstatus wird erreichbar, abwesend, besetzt oder "Nicht stören" angezeigt. Um den Anwesenheitsstatus anzuzeigen, bewegen Sie den Mauszeiger über eine Kontaktkarte. Wenn Sie zum Beispiel telefonieren, wird der entsprechende Anwesenheitsstatus (besetzt) angezeigt. Sie können auch im Dropdown-Menü "Anwesenheit" Ihren Status manuell ändern.
