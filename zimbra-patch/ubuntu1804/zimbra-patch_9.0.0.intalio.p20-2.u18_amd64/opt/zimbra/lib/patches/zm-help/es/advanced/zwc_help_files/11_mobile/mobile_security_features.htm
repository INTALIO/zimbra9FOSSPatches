Funciones de seguridad para móviles

Es posible que tu administrador haya configurado una política de seguridad para móviles que requiera crear un número de identificación personal (PIN) en tu dispositivo móvil. Si olvidas tu PIN, ponte en contacto con el administrador del sistema.

Si dispones de un PIN, es posible que la política esté configurada con una función para bloquear a los usuarios si no introducen el PIN correcto.  El administrador del sistema configura el ajuste de bloqueo del PIN de modo que se lleve a cabo un borrado local del dispositivo si alguien intenta iniciar sesión en un dispositivo con una contraseña incorrecta un número de veces superior al configurado. Esta operación borrará todos los datos del dispositivo. Sólo el administrador puede determinar la configuración de la política que no puede ser modificada por el usuario.  Consulta al administrador del sistema para obtener más información.
