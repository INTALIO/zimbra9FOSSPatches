複数のメッセージを同じ操作にマークする

[内容]ページのチェックボックスを使用し、1つまたは複数のメッセージまたはスレッドを選択し、同じ操作を実行します。

備考：  メールビューにチェックボックスが表示されていない場合は、[プリファレンス]>[全般]ページへ進み、[その他]領域で[設定]を選択します。

ヘッダーのチェックボックスをオンにすると、そのページのすべてのメッセージに自動的にチェックが入ります。

一度に複数のメッセージを削除

  1. [メールの内容] ペインで、削除したいメッセージのチェックボックスをオンにします。

  2. 右クリックし、[削除]を選択します。

複数のメッセージに一度にタグ付け

  1. [メールの内容] ペインで、同じタグを付けたいメッセージのチェックボックスをオンにします。

  2. 右クリックし、[スレッドにタグを付ける]を選択します。

  3. リストからタグを選択するか新しいタグを作成します。

一度に複数のメッセージを印刷

  1. [メールの内容] ペインで、印刷したいメッセージのチェックボックスをオンにします。

  2. 右クリックし、[印刷]を選択します。

  3. 印刷ボックスで、[OK]をクリックします。すべてのメッセージが連続するファイルとして印刷されます。

一度に複数のメッセージを転送

  1. [メールの内容] ペインで、転送したいメッセージのチェックボックスをオンにします。

  2. 右クリックし、[転送]を選択します。メールメッセージが添付された状態でメッセージ作成ページが開きます。

  3. メッセージを入力し、送信します。
