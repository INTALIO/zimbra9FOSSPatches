Zoekopdrachten maken en opslaan

Als je vaak dezelfde zoekopdracht uitvoert, kun je deze opslaan in je zoekmap. Wanneer je de zoekopdracht opnieuw wilt uitvoeren, klik je gewoon op de map met opgeslagen zoekopdrachten, waarna de zoekopdracht wordt uitgevoerd en de resultaten worden weergegeven in het deelvenster Inhoud.

  1. Klik bij de kop Deelvenster Overzicht>Zoeken op Bewerken.  

  2. Selecteer op de werkbalk Nieuwe zoekopdracht.

  3. Voer in het tekstvak Naam je zoekopdracht in.

  4. Selecteer in het tekstvak Hogergelegen map aan waar deze zoekopdracht moet worden opgeslagen.  De map wordt opgeslagen onder de kop Zoeken in het deelvenster Overzicht.  Als je de zoekopdracht wilt opslaan in een andere map, selecteer je die map in de vervolgkeuzelijst. De zoekmap herken je aan het pictogram met het vergrootglas.

  5. Stel in het tekstvak Zoekopdracht de zoekopdracht samen. Zie Beschrijving van de querytaal voor informatie over de zoekgrammatica die je kunt gebruiken.

  6. Klik op Zoeken.

Als je jouw zoekopdracht wilt uitvoeren, klik je op de zoekmap in het deelvenster Overzicht. De zoekopdrachten worden weergegeven in het inhouddeelvenster.
