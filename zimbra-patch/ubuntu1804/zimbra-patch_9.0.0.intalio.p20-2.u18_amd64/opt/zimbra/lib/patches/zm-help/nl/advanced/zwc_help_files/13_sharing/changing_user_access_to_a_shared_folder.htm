Gebruikerstoegang tot een gedeelde map wijzigen

Je kunt de deelrechten die je aan een gebruiker of groep hebt toegewezen, wijzigen.

  1. Klik met de rechtermuisknop op de te wijzigen gedeelde map en selecteer Eigenschappen bewerken. Het dialoogvenster Mapeigenschappen geeft de namen die de map delen weer.

  2. Klik onder Delen voor deze map op Bewerken naast de te wijzigen naam.

  3. Wijzig in het dialoogvenster Eigenschappen gedeeld item de toegangsrechten.

    4. Je kunt de toegang tot een gedeelde map tijdelijk uitschakelen zonder dat daarbij de deelrechten worden ingetrokken door Geen te selecteren.

  4. Geef aan of je de gebruiker wel of niet via een e-mailbericht wilt informeren.

  5. Klik op OK.
