Вложения на незападноевропейских языках

Если при поиске вложения с текстом на незападноевропейском языке это вложение не включено в результаты поиска, оно может быть недоступным для поиска.

Документы на незападноевропейских языках, для которых в заголовке «Тип содержимого» вложения не задан параметр набора символов (charset), могут быть неправильно проиндексированы. Параметром набора символов определяется тип кодировки символов, и таким образом Zimbra определяет, как индексировать содержимое.  Для включения в результаты поиска вложения должны быть правильно проиндексированы. Эта проблема обычно возникает с вложенными файлами в формате обычного текста или в формате .csv.
