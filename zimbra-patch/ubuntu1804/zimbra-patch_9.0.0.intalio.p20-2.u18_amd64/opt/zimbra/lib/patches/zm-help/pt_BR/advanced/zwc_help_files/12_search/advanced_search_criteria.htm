Filtros de busca que você pode usar

Consulte a tabela abaixo para criar buscas mais detalhadas.

Na guia Buscar, você poderá selecionar entre filtros avançados para criar consultas mais complexas.

Filtros avançados

Descrição

Recebido de

Busca certos domínios no cabeçalho da mensagem (De, Para e Cc). Os domínios que você pode buscar são predefinidos.

Enviado para

Busca certos domínios no cabeçalho da mensagem (De, Para e Cc). Os domínios que você pode buscar são predefinidos.

Data de envio

Busca mensagens recebidas após, antes ou em uma data selecionada. Para buscar entre as datas, selecione Data de envio é anterior a, selecione Data de envio é posterior a e insira o intervalo de datas.  

Anexos

Busca e-mail com ou sem anexo. Se você estiver pesquisando um tipo específico de anexo, selecione o tipo na lista de formatos exibidos.

Tamanho

Busca mensagens por tamanho (bytes, KB ou MB).

Status

Busca mensagens ou contatos com base no seu status, como lidas, não lidas, encaminhadas ou respondidas.

Marcador

Busca mensagens ou contatos com um marcador específico.

Pasta

Limita a busca às pastas selecionadas.
