Ortsverfügbarkeit mit der Planungsfunktion anzeigen

Die Planungsfunktion des Kalenders kann Sie dabei unterstützen, die nächsten verfügbaren Besprechungszeiträume und -orte in Ihrer Organisation zu finden. Sie können Angaben zu bevorzugten Zeiten, Gebäuden, Orten und Raumgrößen für eine Besprechung machen.

Während Sie die Namen der Teilnehmer eingeben, schlägt die Planungsfunktion im Übersichtsfenster entsprechende Zeiten und Orte vor. Sie können auch nach einem Besprechungsort suchen, bevor Sie mit dem Hinzufügen von Teilnehmern beginnen. Entsprechende Einstellungen, die Sie vorgenommen haben, werden beim Anzeigen von Orten berücksichtigt.

Ortseinstellungen mit dem Planungswerkzeug festlegen

Wenn Sie für Ihre Besprechungen immer Räume an einem bestimmten Ort reservieren möchten, können Sie die Funktion "Vorgeschlagene Orte" in der Kalenderanwendung verwenden. Die Verfügbarkeit der entsprechenden Räume wird dann beim Erstellen eines Termins berücksichtigt.

  1. Klicken Sie auf der Seite Neuer Termin auf Ort vorschlagen.

  2. Klicken Sie in dem Fenster, das geöffnet wird, auf das Zahnradsymbol.

  3. Geben Sie die Informationen zum Ort im Dialogfeld Einstellungen für Vorschläge im Abschnitt "Ortseinstellungen" ein. Sie können nur Namen des Orts eingeben oder weitere Angaben hinzufügen.

  4. Klicken Sie auf OK.

In der Spalte rechts werden die Orte angezeigt, die Ihren Anforderungen entsprechen. Damit sie angezeigt werden können, müssen die Räume (Orte) im Firmenverzeichnis konfiguriert sein.

Nachdem Sie diese Einstellungen für einen Termin konfiguriert haben, werden die Informationen gespeichert und stehen für weitere Termine zur Verfügung.
