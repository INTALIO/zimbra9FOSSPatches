Configurar preferencias para responder correos

Para responder el mensaje en el mismo formato que como lo recibiste, selecciona Responder/reenviar usando el formato del mensaje original. Esto asegurará que el destinatario reciba el mensaje en el formato más accesible.

  1. Ve a la página Preferencias>Correo, sección Redactar mensajes.

  2. Para responder el mensaje en el mismo formato que como lo recibiste, selecciona Responder/reenviar usando el formato del mensaje original.

  3. Selecciona si quieres incluir el texto original en tu mensaje de respuesta. Si vas a incluir el mensaje original:

    4. •   Selecciona si deseas marcar las respuestas anteriores con un carácter o incluir los detalles originales en el encabezado (los campos "Para", "De" y "Asunto"). También puedes incluir el mensaje original como archivo adjunto.

    •    Si deseas añadir un prefijo a las respuestas anteriores, selecciona el corchete angular de cierre (>) o la barra vertical (|).

  4. Haz clic en Guardar.
