Aufgabe erstellen

Sie legen eine neue Aufgabe an, indem Sie einen Namen für die Aufgabe in Ihrer Aufgabenliste eingeben. Sie können auch eine neue Aufgabenseite öffnen und ausführliche Informationen zu der Aufgabe aufzeichnen, u. a. Notizen und Anhänge.

Rasch eine Aufgabe anlegen

  1. Öffnen Sie die Seite Aufgaben und wählen Sie im Übersichtsfenster die Aufgabenliste aus, der Sie eine Aufgabe hinzufügen möchten.

  2. Geben Sie den Namen der neuen Aufgabe im Textfeld Klicken Sie hier, um eine neue Aufgabe zu erstellen des Inhaltsfensters ein.

  3. Drücken Sie die Eingabetaste. Der Betreff der Aufgabe wird der Aufgabenliste hinzugefügt. Es werden jedoch keine Details aufgezeichnet. Wenn Sie weitere Informationen hinzufügen möchten, bearbeiten Sie die Aufgabe.

Hinweis:   Wenn Sie eine Aufgabe in der falschen Liste angelegt haben, öffnen Sie die Aufgabe und wählen in der Dropdown-Liste "Aufgabenlisten" die richtige Liste aus. Sie können die Aufgabe auch mit der Maus auf die richtige Liste im Inhaltsfenster ziehen.
