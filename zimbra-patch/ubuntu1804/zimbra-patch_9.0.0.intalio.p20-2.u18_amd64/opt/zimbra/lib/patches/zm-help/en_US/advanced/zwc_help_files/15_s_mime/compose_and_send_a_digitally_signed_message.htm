Compose and Send a Digitally Signed Message

Note: This feature is for ZCS Network Edition only and is available only in the Advanced Web Client.

Digitally signed messages can be sent to recipients that have sent you their certificate. The recipientâ€™s certificate is stored in the recipients contact page in Contacts. 

  1. From the Mail tab, click New Message from the toolbar.

  2. In To: enter the user's address or click the To: link to search for an address from you address books.

  3. If your security preference is not set by default to send digitally signed messages, click Security and select Sign only.

  4. Enter a subject of the message.

  5. Compose the message.

  6. To add an attachment, click Add Attachment and browse to the file, or drag and drop the file from your computer to the email header.

  7. Click Send.
