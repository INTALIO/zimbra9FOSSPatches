Création de documents dans le Porte-documents

Vous pouvez créer des documents dans le porte-documents et concevoir la présentation du contenu en utilisant les outils en format RTF disponibles dans Porte-documents.

  1. Accédez à l'onglet Porte-documents et sélectionnez le dossier Porte-documents où vous voulez créer un nouveau document.

  2. Cliquez sur Nouveau Document. Une page vierge apparaît :

  3. Rédigez le contenu de la page à l'aide de l'éditeur de texte enrichi. Vous pouvez ajouter des styles, des couleurs et des tableaux à votre page à partir des liens dans la barre d'outils.

  4. Entrez le nom du nouveau document dans le champ Enregistrer.

  5. Cliquez sur Enregistrer.
