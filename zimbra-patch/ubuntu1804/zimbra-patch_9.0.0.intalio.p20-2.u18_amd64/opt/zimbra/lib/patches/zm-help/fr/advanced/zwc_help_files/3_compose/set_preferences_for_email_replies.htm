Définir des préférences pour les réponses aux mails

Pour répondre au mail sous le même format que vous l'avez reçu, sélectionnez Répondre/Transférer en utilisant le format du mail original. Cela garantit que le destinataire reçoive le mail dans le format le plus lisible.

  1. Allez sur la page Préférences>Mail, section Création de mails.

  2. Pour répondre au mail sous le même format que vous l'avez reçu, sélectionnez Répondre/Transférer en utilisant le format du mail original.

  3. Choisissez d'inclure le texte original dans votre mail de réponse. Si vous incluez le mail d'origine :

    4. •   Choisissez s'il faut marquer les réponses précédentes avec un caractère ou inclure les détails d'origine dans l'en-tête (les champs À, De et Objet). Vous pouvez également inclure le mail d'origine comme pièce jointe.

    •    Si vous voulez un préfixe pour les réponses précédentes, sélectionnez le caractère crochet point (>) ou barre verticale (|).

  4. Cliquez sur Enregistrer.
