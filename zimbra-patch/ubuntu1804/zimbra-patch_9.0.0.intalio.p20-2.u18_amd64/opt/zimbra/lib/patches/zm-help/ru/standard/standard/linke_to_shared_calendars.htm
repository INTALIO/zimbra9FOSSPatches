Подключение к общим ежедневникам

Получив почтовое уведомление о предоставлении общего доступа к ежедневнику другого пользователя, вы должны подключиться к общему ежедневнику, чтобы он появился в вашем списке ежедневников.  Для получения доступа необходимо знать полный электронный адрес владельца и точное имя ежедневника.

Как подключиться к ежедневнику общего доступа.

  1. В области обзора в заголовке «Ежедневник» щелкните Изменить.

  2. На панели инструментов выберите пункт Подключить к общему ежедневнику.

  3. Введите имя общего ежедневника. Это имя, которое будет отображаться в вашем списке ежедневников.

  4. Для получения доступа к ежедневнику необходимо ввести электронный адрес владельца и имя ежедневника.

  5. Выберите цвет для этого общего ежедневника.

  6. Щелкните пункт Создать ежедневник.
