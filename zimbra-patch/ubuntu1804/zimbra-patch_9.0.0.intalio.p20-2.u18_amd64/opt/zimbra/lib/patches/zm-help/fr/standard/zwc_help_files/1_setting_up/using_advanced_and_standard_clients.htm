Utilisation des clients avancés et standard

Deux versions de @ClientNameLong@ sont disponibles.

Le client Web avancé offre un ensemble complet de fonctions de collaboration Web. Il fonctionne avec les navigateurs les plus courants et les connexions Internet haut débit.

Si vous avez une connexion Internet lente ou si vous préférez une messagerie HTML, vous pouvez utiliser la version client Web standard. Elle contient globalement les mêmes fonctions que la version client Web avancé mais vous pouvez y accéder différemment. Vous pouvez changer la valeur par défaut comme client Web standard à partir de votre page Préférences>générales après que vous vous connectez.

Outils de navigation

•Le bouton Retour du navigateur vous permet de revenir à la page précédente que vous consultez.。 Vous pouvez également utiliser le bouton Suivant.

•N’utilisez pas le bouton Recharger (Actualiser) du navigateur. Cela redémarrera votre session.
