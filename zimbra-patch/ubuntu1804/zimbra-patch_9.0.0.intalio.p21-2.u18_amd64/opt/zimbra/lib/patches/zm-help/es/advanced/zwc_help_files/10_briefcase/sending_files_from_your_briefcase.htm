Enviar archivos desde el Maletín

Al trabajar en las carpetas de tu maletín, puedes enviar un enlace a determinados archivos del maletín, y también puedes enviar el archivo en cuestión como un archivo adjunto a un mensaje de correo electrónico. Los destinatarios deberán tener el software apropiado para abrir los archivos.

Enviar un mensaje con un enlace a un archivo

Puedes enviar un mensaje de correo electrónico que incluya un enlace a un archivo del maletín. El destinatario debe tener los permisos para compartir correspondientes si desea trabajar con el archivo, así como el software adecuado para abrirlo. Cualquier cambio que se realice al archivo enlazado se verá reflejado en el archivo en la carpeta de tu maletín.

  1. Ve a la pestaña Maletíny selecciona el archivo que deseas compartir.

  2. Haz clic en Accionesy selecciona Enviar enlaces. Se abrirá un panel de confirmación para recordarte que el destinatario debe tener permiso para ver el contenido de la carpeta. Haz clic en Sípara continuar.

  3. Se abrirá una nueva ventana de redacción de mensaje que incluirá el enlace en el cuerpo. Completa el mensaje y haz clic en Enviarpara enviarlo.

Enviar un archivo por correo electrónico como archivo adjunto

Puedes enviar un archivo como adjunto de un correo electrónico directamente desde el maletín. Los destinatarios deberán tener el software apropiado para abrirlo.

  1. Ve a la pestaña Maletíny selecciona el archivo que deseas adjuntar.

  2. Haz clic en Accionesy selecciona Enviar archivos adjuntos. Se abrirá la página de redacción del mensaje, que incluirá el archivo adjunto.

  3. Añade la dirección de correo electrónico del destinatario y escribe el mensaje.

  4. Haz clic en Enviar.
