新しい連絡先を追加

  1. [アドレスブック]ページへアクセスし、[新しい連絡先]を選択します。

  2. その他の連絡先情報を入力し、オプションとして写真をアップロードします。

    3. テキストボックスの横のプラス記号は、複数のエントリを追加できることを示しています。たとえば、[電話番号]テキストボックスでは、携帯電話、自宅、職場およびその他の電話番号を追加できます。

  3. カスタムテキストボックスを追加するには、テキストボックスのドロップダウンメニューから[その他]を選択し、情報を入力します。

  4. 画面右上で、連絡先を保存する方法と場所を選択します。

    5. •    [保存するファイルの名前]で、名前の保存形式を選択します。デフォルトでは、「姓、名」の形式となっています。

    •    [場所]で、連絡先を保存するアドレスブックを選択します。

  5.  [保存]をクリックします。
