Создание и сохранение запроса поиска

В случае частого выполнения какого-либо запроса поиска можно создать запрос и сохранить его в папке «Поиск». Для повторного выполнения поиска нужно просто щелкнуть папку с сохраненным профилем поиска. Это действие запускает поиск, результаты которого отображаются в области содержимого.

  1. В области Обзор>Поиск нажмите кнопку Изменить.  

  2. На панели инструментов выберите пункт Новый поиск.

  3. В текстовом поле Имя укажите имя профиля поиска.

  4. В текстовом поле Родительская папка укажите, где необходимо сохранить этот профиль поиска.  Папка будет сохранена под заголовком «Поиск» в области обзора.  Если профиль поиска необходимо сохранить в другой папке, выберите нужную папку в раскрывающемся списке. Папку поиска можно узнать по значку с изображением лупы.

  5. В текстовом поле Запрос поиска создайте запрос поиска. Сведения о грамматике языка поиска см. в разделе Описание языка запросов.

  6. Нажмите кнопку Поиск.

Для выполнения поиска щелкните папку поиска в области обзора. Результаты поиска будут показаны в области содержимого.
