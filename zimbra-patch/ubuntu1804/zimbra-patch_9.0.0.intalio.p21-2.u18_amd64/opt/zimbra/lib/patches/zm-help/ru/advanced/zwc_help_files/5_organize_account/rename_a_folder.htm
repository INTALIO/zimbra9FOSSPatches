Переименование папки

Можно переименовывать папки, определяемые пользователем, но не системные папки.

  1. Перейдите на страницу Почта>Папки.

  2. Щелкните правой кнопкой мыши папку и выберите пункт Переименовать папку. 

  3. Введите новое имя папки.

    4. Имена папок могут содержать любые символы, кроме двоеточия (:), косой черты
    (/) и кавычки (").

  4. Нажмите кнопку ОК.
