Criar e salvar um critério de busca

Se você usar o mesmo critério de busca com freqüência, convém salvá-lo na pasta Busca. Quando você quiser fazer a busca novamente, basta clicar na pasta de busca salva para executá-la, exibindo os resultados no painel Conteúdo.

  1. No painel Visão geral>cabeçalho Busca, clique em editar.  

  2. Na barra de ferramentas, selecione Nova busca.

  3. Na caixa de texto Nome, dê um nome para sua busca.

  4. Na caixa de texto Pasta principal, selecione onde salvar a busca.  A pasta é salva no cabeçalho Busca, painel Visão geral.  Se você quiser salvar a busca em outra pasta, selecione a pasta em questão na lista. É possível identificar a pasta de busca pelo ícone de aumento.

  5. Na caixa de texto Critério de busca, crie o seu critério. Consulte Descrição da linguagem de critérios para obter informações sobre a sintaxe que pode ser usada.

  6. Clique em Buscar.

Para executar sua busca, clique na pasta desejada no painel Visão geral. Os resultados da busca são exibidos no painel Conteúdo.
