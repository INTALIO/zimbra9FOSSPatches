Применение одного и того же действия к нескольким сообщениям

С помощью флажков на странице содержимого можно выбрать одно или несколько сообщений или разговоров, чтобы выполнить с ними одинаковое действие.

Примечание.   Если флажки не отображаются в виде «Почта», перейдите на страницу «Настройки>Общие» и в разделе «Другое» выберите «Параметры».

Если установить флажок в заголовке, автоматически будут выбраны все сообщения на странице.

Удаление нескольких сообщений одновременно

  1. В области содержимого установите флажки для сообщений, которые следует удалить.

  2. Щелкните правой кнопкой мыши встречу и выберите пункт Удалить.

Пометка тегом нескольких сообщений

  1. В области содержимого установите флажки для сообщений, которые следует пометить одинаковым тегом.

  2. Щелкните правой кнопкой мыши и выберите пункт Пометить разговоры.

  3. Выберите тег из списка или создайте новый тег.

Печать нескольких сообщений одновременно

  1. В области содержимого установите флажки для сообщений, которые следует распечатать.

  2. Щелкните правой кнопкой мыши и выберите пункт Печать.

  3. В окне печати нажмите кнопку ОК. Все сообщения будут напечатаны как один непрерывный файл.

Пересылка нескольких сообщений одновременно

  1. В области содержимого установите флажки для сообщений, которые следует переслать.

  2. Щелкните правой кнопкой мыши и выберите пункт Переслать. Откроется страница создания сообщения с вложенными сообщениями электронной почты.

  3. Заполните и отправьте сообщение.
