Recherche de mails envoyés à un nom de contact ou reçus de celui-ci

Vous pouvez rechercher tous les mails reçus d'un nom de contact spécifique ou envoyés à celui-ci dans votre carnet d'adresses.

Lorsque vous effectuez la recherche dans votre carnet d'adresses, les résultats de tous les mails de tous les dossiers de votre compte s'affichent.

  1. Sélectionnez un nom de contact dans votre carnet d'adresses.

  2. Cliquez avec le bouton droit sur le message et sélectionnez Rechercher les mails.

  3. Choisissez Reçu du contact ou Envoyé au contact. La recherche est exécutée et tous les mails de votre compte qui correspondent à votre requête s'affichent dans le Panneau de contenu.
