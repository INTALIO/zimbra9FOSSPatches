Lecture d'un mail

Par défaut, les nouveaux mails sont placés dans votre boîte de réception. Si vous avez configuré des règles de filtre de mails, les nouveaux messages peuvent apparaître dans d’autres dossiers que votre boîte de réception. Le nombre entre parenthèses à côté de chaque dossier correspond au nombre de mails non lus dans ce dossier.

Lire un mail dans l'affichage Conversation

  1. Pour lire un mail dans la vue de conversation, double-cliquez sur la conversation contenant le mail. Le panneau de contenu change et affiche les mails individuels de la conversation.

    2. •    Pour développer le message afin de voir toutes les conversations, cliquez sur Afficher le texte entre guillemets à la fin du message.

    •    Pour réduire la fenêtre de conversation afin de n'afficher que le dernier message, cliquez sur Masquer le texte entre guillemets à la fin du message.

    •    Pour réduire tous les messages afin de n'afficher que le nom de l'expéditeur et l'extrait de la première ligne du message, double-cliquez sur l'en-tête du message.

  2. Pour ouvrir le mail, cliquez sur la ligne qui contient le mail à lire.

Si le panneau de lecture est activé, le corps du mail apparaît dans ce panneau. Sinon, double-cliquez sur le mail. Le mail s'affiche dans le panneau de contenu.

Lire un mail dans l'affichage Message simple

•       Si la fonction Panneau de lecture est activée, cliquez sur le message, puis lisez-le dans le panneau de lecture.

•       Si la fonction Panneau de lecture est désactivée, double-cliquez sur le message. Le mail s'affiche dans le panneau de contenu.
