Suchabfrage erstellen und speichern

Wenn Sie häufig dieselbe Suchabfrage durchführen, können Sie eine Suchabfrage erstellen und diese in Ihrem Suchordner speichern. Wenn Sie die Suche erneut durchführen möchten, klicken Sie einfach auf den Ordner mit den gespeicherten Suchvorgängen, um die Suche zu starten. Die Ergebnisse werden im Inhaltsfenster angezeigt.

  1. Klicken Sie unter Übersichtsfenster>Suche auf Bearbeiten.  

  2. Klicken Sie in der Symbolleiste auf Neue Suche.

  3. Geben Sie im Textfeld Name einen Namen für die Suche ein.

  4. Wählen Sie im Textfeld Übergeordneter Ordner aus, wo diese Suche gespeichert werden soll. Der Ordner wird im Übersichtsfenster unter der Überschrift "Suchen" gespeichert. Wenn Sie die Suche in einem anderen Ordner speichern möchten, wählen Sie den Ordner in der Dropdown-Liste aus. Sie können den Suchordner am Symbol mit dem Vergrößerungsglas erkennen.

  5. Erstellen Sie im Textfeld Suchabfrage Ihre Suchabfrage. Informationen zur zulässigen Suchgrammatik finden Sie unter Beschreibung der Sprache für Suchabfragen.

  6. Klicken Sie auf Suchen.

Zum Ausführen Ihrer Suche klicken Sie im Übersichtsfenster auf den Suchordner. Die Suchergebnisse werden im Inhaltsfenster angezeigt.
